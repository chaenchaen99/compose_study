### 📅 프로젝트 개요

- **개발 기간**: 2024.11 - 2024-12 
- **사용 기술**: 
  - Android Studio
  - Kotlin
  - Retrofit
  - Hilt
  - Navigation
  - Clil

### 🚀 주요 기능

- 영화리스트 불러오기 및 표시
- 영화디테일 화면 표시


### 🚀 프로젝트를 통해 배운 점
- CompositionLacal 활용
- Data class를 활용한 컴포넌트의 동적 활용
- Jetpack Compose 기반 Design System 적용
